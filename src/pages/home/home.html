Building Semantic Ontology Project

is an open-source initiative aimed to ease the process of tagging data in buildings based on Project Haystack.

The goal of the project is to
  • To define a structure of all the elements used in a BMS
  • To define types to define multi-tags entities
  • To handle the translations so the user uses its own language to do the tagging while keeping standardised tags
